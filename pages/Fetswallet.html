Fets Wallet

Fets Wallet is a financial technology website that simplifies financial transactions for users. It provides a secure platform for managing funds, making payments, and accessing a range of financial services.

Role

UX/UI Designer


Team

1 Product Owner | 2 Stakeholder Representatives | 1 UX Researcher | Developers | QAs | Consumer Service Reps | Scrum Masters | Marketing Specialists


The Problem

FETS WALLET is a financial technology website that allows users to carry out value-added services. However, the platform was facing several challenges that made it difficult for users to complete transactions easily and efficiently. Some of the key problems included:

Lack of Clarity: The platform lacked clear instructions, making it challenging for users to complete transactions without confusion.

Poor User Experience: The platform was not user-friendly, with complex features and too many options that overwhelmed users.

Security Concerns: Users were concerned about the safety of their financial information and transactions, which led to low user engagement and trust.

The User Experience

To ensure a positive user experience, we conducted extensive user research and testing, incorporating feedback from users to guide our design decisions. Some of the key design decisions that contribute to a positive user experience include:

A Minimalist Design: The new platform has a minimalist design that reduces clutter and distractions, making it easier for users to focus on the task at hand.

Clear Feedback Messages: The new platform provides clear and concise feedback messages that help users avoid errors and understand the transaction process.

Easy Navigation: The new platform has a simple navigation structure that allows users to move between pages quickly and easily, with clear labels and visual cues that guide them along the way.

The Solution

To address these challenges, we designed a new FETS WALLET platform that focused on improving user experience, clarity, and security. Some of the key features of the new platform include:

A User-Friendly Interface: The new platform has a simple and intuitive interface that is easy to navigate and understand, with clear instructions and feedback messages that help users complete transactions efficiently.

Enhanced Security Features: The new platform has robust security features that protect user information and transactions, including two-factor authentication and encryption.

Streamlined Transactions: The new platform has a streamlined transaction process that allows users to complete transactions quickly and easily, without the need for multiple steps or complicated features.

Process

To design the new FETS WALLET platform, we followed a user-centered design process, incorporating user feedback and insights at every stage of the design process. Our design process included:

User Research: We conducted extensive user research to understand user needs, pain points, and preferences.
Design Iteration: We iterated on the design, incorporating user feedback and testing at every stage.
User Testing: We conducted user testing to validate our design decisions and ensure a positive user experience.
Implementation: We implemented the design and monitored user engagement and feedback to make any necessary adjustments.
Overall, our design process focused on creating a user-friendly, secure, and efficient platform that meets the diverse needs and preferences of FETS WALLET users.
